# C39 · Crunch Project Management — Brand Guide

- **Accent:** Code Crunch blue `#2463EB` (with deep `#1d4ed8`).
- **Voice:** calm, evidence-driven, anti-theater. Always show the plan, then the metric, then *why* the delivery moved the way it did. No process for process's sake — every ceremony and artifact earns its place or gets cut. Data over vibes: measure flow with SQL + Python, never a spreadsheet-as-database.
- **Icon motif:** a kanban board column + a burndown line.
- **One-liner:** "Ship on time without the theater — Agile delivery, measured and humane."
