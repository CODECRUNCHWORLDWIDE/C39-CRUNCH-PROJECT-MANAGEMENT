# Week 2 — Exercises

Three guided reps that turn this week's lectures into reflexes: design a real sprint cadence, set defensible WIP limits from a query instead of a guess, and facilitate a retro that actually produces action items. Work them in order — each assumes the vocabulary and, where noted, the seed data from the lecture it follows.

## How to work these

- Exercises 2 uses the `atlas_backlog` table from this week's [README setup](../README.md#set-up-this-weeks-data-do-this-first) — make sure it's seeded before you start.
- Exercises 1 and 3 are facilitation/writing deliverables — no database needed, but they still expect concrete, specific answers, not vague gestures at "communicate better."
- Create a folder in your own portfolio repo, `c39-week-02/`, and put each exercise's output in its own subfolder (`exercise-01/`, `exercise-02/`, `exercise-03/`) as instructed in each file.
- These are self-checked: each exercise gives an expected outcome or a way to verify your own answer. Compare honestly before moving on.
- If you get stuck, re-read the relevant lecture section named in the exercise rather than guessing.
