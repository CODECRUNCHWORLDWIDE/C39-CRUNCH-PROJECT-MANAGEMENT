# Week 2 — Challenges

Two harder, more open-ended problems that ask you to apply this week's material to messier situations than the exercises gave you. Where the exercises had a clear right answer to check against, these expect a written, defended judgment call — the kind you'll actually make on a real team.

## How to work these

- Each challenge is a written deliverable in your portfolio: create `c39-week-02/challenge-01/` and `c39-week-02/challenge-02/` as instructed in each file.
- These are harder than the exercises on purpose. Expect to spend real time thinking before you write.
- There isn't always one right answer — but there are wrong reasoning patterns. Each challenge tells you what a strong answer needs to contain and cites the lecture sections a strong answer draws on.
