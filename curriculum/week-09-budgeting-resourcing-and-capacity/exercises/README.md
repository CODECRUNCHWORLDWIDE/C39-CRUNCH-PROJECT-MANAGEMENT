# Week 9 — Exercises

Three guided reps that turn this week's lectures into reflexes. Work them in order — each assumes the vocabulary and the `atlas_pm` tables from the lecture it follows.

## How to work these

- Exercises 1–3 use the `people`, `budget_lines`, `actuals`, `time_entries`, and `allocations` tables from the [week README](../README.md) setup and the data inserted across Lectures 1–3. Have `atlas_pm` running, with all three lectures' `INSERT` statements applied, before you start.
- Create a folder in your own portfolio repo, `c39-week-09/`, and put each exercise's output in its own subfolder (`exercise-01/`, `exercise-02/`, `exercise-03/`) as instructed in each file.
- These are self-checked: each exercise gives you an expected outcome or a way to verify your own answer. Compare honestly before moving on.
- If you get stuck, re-read the relevant lecture section named in the exercise rather than guessing — the goal is accuracy, not speed.
