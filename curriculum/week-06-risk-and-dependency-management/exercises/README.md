# Week 6 — Exercises

Three short, guided reps that turn this week's lectures into reflexes: score risk consistently, choose responses without dodging, and compute a critical path without guessing.

## How to work these

- Exercises 1 and 2 use the `risks` table you created in the week's setup (PostgreSQL or SQLite) — you'll be writing real `INSERT` and `SELECT`/`UPDATE` statements, not filling in a document.
- Exercise 3 uses Python (`pandas`) — reuse the forward-pass/backward-pass script from Lecture 3, §6 as your starting point rather than rewriting it from scratch.
- Create a folder in your own portfolio repo, `c39-week-06/`, and put each exercise's output in its own subfolder (`exercise-01/`, `exercise-02/`, `exercise-03/`) as instructed in each file.
- These are self-checked: each exercise gives you an expected outcome or a way to verify your own answer. Compare honestly before moving on.
- If you get stuck, re-read the relevant lecture section named in the exercise rather than guessing — the goal is accuracy, not speed.
