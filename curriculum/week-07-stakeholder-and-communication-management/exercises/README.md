# Week 7 — Exercises

Three short, guided reps that turn this week's lectures into reflexes. Work them in order — each assumes the vocabulary and the `atlas_pm` tables from the lecture it follows.

## How to work these

- Exercises 1–3 use the `stakeholders`, `comms_plan`, and `status_log` tables from the [week README](../README.md) setup. Have `atlas_pm` running before you start.
- Create a folder in your own portfolio repo, `c39-week-07/`, and put each exercise's output in its own subfolder (`exercise-01/`, `exercise-02/`, `exercise-03/`) as instructed in each file.
- These are self-checked: each exercise gives you an expected outcome or a way to verify your own answer. Compare honestly before moving on.
- If you get stuck, re-read the relevant lecture section named in the exercise rather than guessing — the goal is accuracy, not speed.
