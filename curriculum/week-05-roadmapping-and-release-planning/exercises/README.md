# Week 5 — Exercises

Three guided reps that turn this week's lectures into reflexes: build a roadmap, sequence a release plan, and re-plan after a slip. Work them in order — each assumes the vocabulary and the Northlight Q3 dataset from the lecture it follows.

## How to work these

- Exercises 1–3 use SQL and/or Python against the `roadmap_items` / `sprints` schema introduced in Lecture 2, §2. Set that schema up once (PostgreSQL or SQLite) before starting Exercise 1.
- Create a folder in your own portfolio repo, `c39-week-05/`, and put each exercise's output in its own subfolder (`exercise-01/`, `exercise-02/`, `exercise-03/`), as instructed in each file.
- These are self-checked: each exercise gives an expected outcome or a way to verify your own answer. Compare honestly before moving on — several of these deliberately change one input from the lecture's worked example (a different capacity number, a different slip), so a correct answer will **not** be an exact copy of the lecture's numbers. If yours matches the lecture exactly, re-read the exercise — you likely missed the changed input.
- If you get stuck, re-read the relevant lecture section named in the exercise rather than guessing — the goal is accurate sequencing, not speed.
