# Week 5 — Challenges

Two harder, more open-ended problems. Where the exercises had a spot-checkable expected result, these have a rubric — there's a range of defensible answers, and the rubric tells you what separates a weak one from a strong one.

## How to work these

- Do these after all three exercises — both challenges assume you're fluent with the Northlight Q3 dataset and comfortable running the SQL/Python from Lecture 2.
- Budget real time. These aren't 20-minute problems; they're closer to the weight of a real Tuesday-afternoon roadmap crisis.
- There's no single right answer, but there is a wrong shape of answer: vague reasoning, unstated judgment calls, or numbers that don't reconcile with the dataset will all cost you against the rubric.
